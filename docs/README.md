# docs

This folder contains documentation for the GroundedOS Lab project. At present, it focuses on AI concept explanations and guided study tracks, with architecture decisions and hands-on tutorials planned for future additions.

## Responsibilities

- Explain core AI concepts used throughout the project
- Map concepts into guided learning routes across packages, experiments and roadmap phases
- Provide a foundation for future architecture and tutorial documentation
- Serve as the single source of truth for project knowledge
